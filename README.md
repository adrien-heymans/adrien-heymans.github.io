Personal portfolio site for Adrien Heymans — Enterprise Architect, AI practitioner, and automation builder based in Ottawa, Canada 🍁, originally from Brussels 🇧🇪.

Built as a single HTML file with a **claymorphism** design — frosted glass cards, soft pastel gradients, animated blobs, and zero dependencies. Deployed on GitHub Pages.

---

## 🎨 Design

Claymorphism aesthetic: `backdrop-filter` frosted glass, soft pastel palette, pill-shaped buttons, animated gradient background blobs, fully responsive. Fonts: [Syne](https://fonts.google.com/specimen/Syne) + [Nunito](https://fonts.google.com/specimen/Nunito).
